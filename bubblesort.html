Bubble Sort

Description

Bubble Sort is a comparison-based algorithm in which each pair of adjacent elements is compared and the elements are swapped if they are not in order. This process is repeated until no more swaps are needed, which means the list is sorted. The algorithm gets its name because smaller elements "bubble" to the top of the list (beginning of the array) with each iteration.

This algorithm performs its task by iterating over a list, comparing adjacent elements, and swapping them if they are not in the correct order. The process continues until the list is sorted, with larger elements "bubbling" to the end of the list with each iteration.

  • How It Works: Starting at the beginning of the list, each pair of adjacent elements is compared and swapped if necessary. This process is repeated for the entire list, causing larger elements to bubble to the end of the list.
  • Time Complexity: Best Case - O(n), Average and Worst Case - O(nÂ²). The efficiency decreases as the size of the list increases.
  • Usage Scenarios: Primarily used for educational purposes and for sorting small datasets where simplicity is preferred over efficiency.
  • Advantages and Disadvantages: While it is simple to understand and implement, and efficient for small lists, it is inefficient for larger lists with a significant number of elements.
